Test d'égalité

Sur cette page, je vais présenter l'élité Java avec == et equals.

En effet beaucoup de personne se trouve confonter à ce test car suivant certain cas l'égalité se fait pas comme prévue.

Commençons d'abord par exemple avec 2 chaîne String.
