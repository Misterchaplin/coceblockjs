Error MSB3428

Dans plusieurs cas, quand on utilise npm qui est un gestionnaire de dépendances, on peut se trouver confronté à l'erreur suivant:

MSBUILD : error MSB3428: Impossible de charger le composant Visual C++ "VCBuild.exe". Pour corriger le problème, vous devez :

  • 1) installer le Kit de développement .NET Framework 2.0 SDK
  • 2) installer Microsoft Visual Studio 2005
  • 3) ajouter l'emplacement du composant au chemin d'accès système, s'il est installé ailleurs. Chemin dans votre projet

Cette erreur signifie qu'il manque des outils microsoft. Cela veut aussi dire que vous n'avez surement pas le logiciel Visual Studio d'installer sur votre machine.

La façon la plus simple de résoudre ce problème, est de tout simplement de télécharger Microsoft Visual Studio et de l'installer (Environ 8 go). Ensuite vous pourrez répéter votre commande npm et cela est censé fonctionner.
